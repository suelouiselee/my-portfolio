Asemic Poetry Generator

Home | Back to Projects

Uses random walk algorithms and the RiTa library to generate writing and display them as visually abstracted text points on a canvas. The RiGrammar object processes a predefined poem structure from a JSON file to create randomized poems. A timer triggers the regeneration of writing while functions like circleMagic and createStutter generate abstract geometric patterns when specific keywords are present in the poem.

+ generative art, Natural Language Processing(NLP)

+ P5.js, Javascript, Rita.js
